# IBEXRadar PRO — Frontend

Analizador técnico con STC, SMI, HARSi y alertas Telegram. PWA instalable en Android.

## Stack
- React 18 + Vite
- vite-plugin-pwa (instalable en Android)
- Backend: https://ibexradar-backend.onrender.com

## Desarrollo local

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
```

## Despliegue en Render

- **Build Command:** `npm install && npm run build`
- **Publish Directory:** `dist`
- **Node Version:** 18+
